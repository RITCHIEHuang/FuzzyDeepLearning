# 循环神经网络家族

在序列模型中，RNN 大显身手，NLP 领域简直就是 RNN 的天堂。

听说在股票、基金这种量化交易里也大有用武之地，学一波，说不定能帮助自己走上富贵（亏本）的投资路。

不过 RNN 家族最近发展得有点庞大，我想先从 Vanilla RNN 开始，然后就走向它的改进版 LSTM 和 GRU；
